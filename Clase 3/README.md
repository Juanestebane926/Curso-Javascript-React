# Clase 3 - Un solo Codesandbox

Este proyecto esta organizado en carpetas para evitar cambiar de sandbox durante la clase.

## Estructura
- `index.html`: pagina principal con acceso a los dos momentos.
- `Explicacion/`: demo del docente.
- `Actividad/`: practica para estudiantes.

## Flujo recomendado
1) Abre la pagina principal y entra a `Explicacion/` para programar con el grupo.
2) Luego pasa a `Actividad/` para que completen la practica.

## Nota para CodeSandbox
Si quieres que el proyecto arranque en una vista principal, usa el `index.html` de la raiz.
