# Clase 3 - Actividad PokeAPI

Esta carpeta contiene la actividad para entregar a los estudiantes en CodeSandbox.
La idea es que el ejercicio sea corto, claro y enfocado solo en PokeAPI.
Aunque la respuesta real de la API tiene muchos campos, en esta actividad solo
vamos a trabajar con un objeto simple: id, nombre e imagen.

## Archivos
- index.html
- styles.css
- app.js

## Uso en clase
1) Tu explicas y programas la idea general.
2) Luego les compartes esta actividad para que completen los TODO.
3) Mientras ellos trabajan, los ayudas a resolver errores y a entender el flujo.

## Lo que practican
- Eventos con botones e input.
- Fetch con async/await.
- Reducir una respuesta grande de API a un objeto simple.
- Renderizar informacion de una API.
- Buscar por numero de la Pokedex.
- Mostrar un pokemon aleatorio.
